403 Forbidden vs 404 Not Found

403 denies access to known resource; 404 says resource not found.

Aspect403 Forbidden404 Not Found
When it happensPrimary condition specific to 403 forbidden.Primary condition specific to 404 not found.
Troubleshooting first stepCheck logs and config where 403 is generated.Check logs and config where 404 is generated.
Client impactDifferent retry behavior expected.Different retry behavior expected.

When each one appears

Use response semantics accurately so clients, CDNs, and crawlers make correct decisions.

How to troubleshoot each

Trace request through edge, gateway, app, and data layers. Confirm whether failure is auth, policy, overload, upstream timeout, or origin crash.
