401 Unauthorized vs 403 Forbidden

401 means authentication is missing/invalid; 403 means authenticated but not allowed.

Aspect401 Unauthorized403 Forbidden
When it happensPrimary condition specific to 401 unauthorized.Primary condition specific to 403 forbidden.
Troubleshooting first stepCheck logs and config where 401 is generated.Check logs and config where 403 is generated.
Client impactDifferent retry behavior expected.Different retry behavior expected.

When each one appears

Use response semantics accurately so clients, CDNs, and crawlers make correct decisions.

How to troubleshoot each

Trace request through edge, gateway, app, and data layers. Confirm whether failure is auth, policy, overload, upstream timeout, or origin crash.
