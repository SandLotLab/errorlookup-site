500 Internal Server Error vs 503 Service Unavailable

500 is unexpected failure; 503 is temporary overload/maintenance.

Aspect500 Internal Server Error503 Service Unavailable
When it happensPrimary condition specific to 500 internal server error.Primary condition specific to 503 service unavailable.
Troubleshooting first stepCheck logs and config where 500 is generated.Check logs and config where 503 is generated.
Client impactDifferent retry behavior expected.Different retry behavior expected.

When each one appears

Use response semantics accurately so clients, CDNs, and crawlers make correct decisions.

How to troubleshoot each

Trace request through edge, gateway, app, and data layers. Confirm whether failure is auth, policy, overload, upstream timeout, or origin crash.
